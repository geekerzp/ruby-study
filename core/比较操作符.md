操作符        含义
================================================================================
==            测试值相等与否
===           用来比较case语句的目标和每个when从句的项
<=>           通用比较操作符，根据接受者小于，等于或者大于其参数，返回1，0 或 -1
<, <=, >=, >  比较大小
=~            正则表达式模式匹配操作符
eql?          如果接受者和参数有相同的类型和相等的值，则返回真。
              1==1.0 => true 1.eql? 1.0 => false
equal?        如果接受者和参数具有相同的对象ID，则返回真。
================================================================================
